## 任务06_03  年龄计算 

**问题描述：**

有五个学生坐在一起，问第五个人多少岁？答比第四个大2岁，第四个人说他比第三个人大2岁，...，第一个人说他10岁。请写一个程序求第5个人的年龄，要求使用递归函数求第5个人的年龄。

**程序输入：**

    无。

**程序输出：**

    输出第5个人的年龄。

**输入示例：**

    无。

**输出示例：**

    18

```c
/*
	说明：改作业是任务06_3 年龄计算
	姓名：付子旺
	学号：409
	班级：2班
	日期：2017年6月5日
*/
#include <stdio.h>
#include <stdlib.h>

int age(int x)
{
	int s;
	if (x==1)
		return 10;
	else
		s=age(x-1)+2;
	return s;
}
int main(void)
{
	int s;
	s=age(5);
	printf("%d\n",s);
	system("pause");
	return 0;
}
```
